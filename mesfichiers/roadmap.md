# ROADMAP — YABISSO BUSINESS

## Phases principales
PHASE 1 : SETUP + UI AUTH + DASHBOARD
PHASE 2 : APPS CORE (Kassa, Employé, Mon App)
PHASE 3 : APPS ADMIN (Dashboard, CRM)
PHASE 4 : APPS METIER (Hotel, Restaurant, École)
PHASE 5 : APPS COMPTABILITÉ (Facture, Compta, Dépense)
PHASE 6 : POLISH + TESTS + DEPLOIEMENT

## Détail par phase

### Phase 1 — Setup + UI Auth + Dashboard
- Setup React Native + Expo + WatermelonDB
- UI: Welcome, Language, Signup, Login
- Dashboard avec navigation vers les 11 apps
- Auth offline-first

### Phase 2 — Apps Core
- **Yabisso Kassa** : Terminal de caisse
- **Employé** : Gestion des employés
- **Mon App** : Profil utilisateur

### Phase 3 — Apps Admin
- **Yabisso Admin Dashboard** ("Travail") : Dashboard admin
- **CRM** : Relation client

### Phase 4 — Apps Métier
- **Yabisso Hotel** : Gestion hôtelière
- **Restaurant** : Gestion restaurant
- **École** : Gestion scolaire

### Phase 5 — Apps Comptabilité
- **Facture** : Gestion factures
- **Compta** : Comptabilité
- **Dépense** : Gestion dépenses

### Phase 6 — Polish
- Tests
- Optimisation
- Déploiement

## État actuel
